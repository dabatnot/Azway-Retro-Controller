Detailed Description

Source file for managing an LED using FreeRTOS tasks on an Arduino-compatible microcontroller.

This file contains the implementations of the functions required to manage an LED connected to a GPIO pin. The LED can be in one of four states: OFF, READY, WAITING, or CONFIG, each causing different behavior in the LED. The management of the LED state is handled by FreeRTOS tasks to ensure efficient multitasking.
