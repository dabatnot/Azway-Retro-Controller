\doxysubsection{Detailed Description}
Header file to manage bitmap creation and display on the OLED screen. 

This file contains functions to draw various bitmaps and screens on an OLED display using the HT\+\_\+\+SSD1306\+Wire library. The functions include drawing logos, loading screens, and different status screens.
